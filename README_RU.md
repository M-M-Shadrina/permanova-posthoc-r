## Описание

Автоматизированный Python-пайплайн для обработки данных количественной МРТ-сегментации мозга по группе испытуемых. Пайплайн выполняет перемаппинг меток атласа, разделение серого и белого вещества, уточнение границ коры, удаление очагов демиелинизации и извлечение метрик для каждой структуры (средняя интенсивность МРТ и объём).

Пайплайн полностью интерактивен — редактировать код вручную не нужно. Оба скрипта запускаются из консоли с пошаговыми подсказками.

---

## Возможности

- Автоматическая пакетная обработка нескольких папок испытуемых
- Гибкое распознавание файлов через glob-паттерны (обрабатывает вариации в именах)
- Перемаппинг меток атласа и разделение ГМ/БМ
- Уточнение границ коры с помощью маски тканевой сегментации
- Опциональное удаление очагов демиелинизации
- Извлечение средней интенсивности МРТ и объёма (мл) для каждой структуры
- Экспорт результатов в `.xlsx` (2 листа)
- Тестовый режим: запуск и проверка результатов на одном испытуемом перед полной обработкой

## Формат входных данных

Каждый испытуемый должен иметь отдельную папку внутри директории с данными. Ожидаемые файлы:

| Файл | Описание | Обязательный |
|------|----------|--------------|
| `JHU_MNI_SS_WMPM_Type-III_MPF*.nii(.gz)` | Файл атласной сегментации мозга с 44 размеченными структурами | ✅ Да |
| `MPFuncor_coef_trim_seg.nii(.gz)` | Маска тканевой сегментации (серое/белое вещество) | ✅ Да |
| `MPFuncor_coef_transform.nii(.gz)` | Исходная карта количественной МРТ (например, MPF) | ✅ Да |
| `lesions.nii(.gz)` | Бинарная маска очагов демиелинизации | ⬜ Опционально |

Пример структуры входных данных:

```
input_data/
├── subject_01/
│   ├── JHU_MNI_SS_WMPM_Type-III_MPF_Ranz_Sha.nii.gz
│   ├── MPFuncor_coef_trim_seg.nii.gz
│   ├── MPFuncor_coef_transform.nii
│   └── lesions.nii.gz
├── subject_02/
│   └── ...
```

> Пайплайн использует гибкое сопоставление имён файлов, поэтому незначительные отличия в названиях обрабатываются автоматически.

## Как запустить

1. Скачайте репозиторий или оба скрипта
2. Запустите основной скрипт из терминала:

```bash
python process_segmentation.py
```

3. Следуйте подсказкам в консоли:

```
Введите путь к папке с данными:       <папка с подпапками испытуемых>
Введите путь к файлу скрипта:         <путь к segmentation_script.py>
Введите путь для сохранения:          <папка для результатов, или Enter рядом с данными>
```

4. Выберите режим:

```
ВЫБЕРИТЕ РЕЖИМ:
1 - Тест на одной папке
2 - Обработать все папки
```

> **Подсказка для Windows:** Можно вставлять пути напрямую из Проводника через «Копировать как путь» — кавычки убираются автоматически.

## Шаги пайплайна

Для каждого испытуемого `segmentation_script.py` выполняет следующие шаги:

| Шаг | Описание |
|-----|----------|
| 1 · Перемаппинг меток | Изменяет порядок меток атласа для выравнивания нумерации подкорковых и корковых структур |
| 2 · Разделение ГМ / БМ | Разделяет атлас на серое вещество (метки 1–19) и белое вещество (метки 20+) |
| 3 · Обработка маски | Перекодирует маску тканевой сегментации в бинарные классы ГМ/БМ |
| 4 · Уточнение границ коры | Умножает атлас БМ на маску тканей для уточнения корковых областей |
| 5 · Объединение сегментаций | Объединяет структуры ГМ и уточнённую кору в единую сегментацию |
| 6 · Удаление очагов | Обнуляет воксели очагов демиелинизации в финальной сегментации (пропускается при отсутствии файла) |

После обработки всех испытуемых `process_segmentation.py` извлекает метрики и сохраняет результаты в Excel.

## Результаты

### Промежуточные файлы по каждому испытуемому

В папке каждого испытуемого появятся следующие файлы:

| Файл | Описание |
|------|----------|
| `remapped.nii.gz` | Атлас с изменённым порядком меток |
| `remapped_GM.nii.gz` | Только структуры серого вещества |
| `remapped_WM.nii.gz` | Только структуры белого вещества |
| `processed_mask.nii` | Перекодированная маска тканей |
| `segmented_cortex.nii` | Атлас БМ, уточнённый маской тканей |
| `combined_result.nii.gz` | Финальная сегментация для извлечения метрик |

### Итоговая таблица

В папку результатов сохраняется файл `results_YYYYMMDD_HHMMSS.xlsx` с двумя листами:

| Лист | Содержимое |
|------|------------|
| `Mean_Intensity` | Средняя интенсивность МРТ по каждой структуре для каждого испытуемого |
| `Volume_mL` | Объём (мл) по каждой структуре для каждого испытуемого |

В обоих листах: **строки = испытуемые**, **столбцы = структуры мозга**.  
Служебные метки (Label 30, Label 60) исключены из результатов.
